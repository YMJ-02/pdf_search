# 📚 PDF Search v2.0 - 사용 가이드

## 🎯 빠른 시작

**PDF_Search_v2.0.exe** 를 다운로드하여 바로 사용해보세요!

### ⚡ **즉시 시작하기**
1. **PDF_Search_v2.0.exe** 더블클릭으로 실행
2. PDF 파일들을 드래그 앤 드롭으로 추가
3. 검색창에 찾고 싶은 내용 입력
4. **"검색 시작"** 버튼 클릭
5. 결과 카드를 클릭하면 해당 PDF가 자동으로 열림

### 2️⃣ **PDF 파일 추가하기**

#### **방법 1: 드래그 앤 드롭 (추천)**
1. Windows 탐색기에서 PDF 파일들을 선택
2. 앱의 "📁 PDF 파일을 여기로 드래그하세요" 영역으로 끌어옴
3. 파일들이 자동으로 추가되고 처리가 시작됨

#### **파일 처리 과정**
- **텍스트 추출**: PDF에서 텍스트를 자동 추출
- **AI 분석**: 각 페이지를 AI가 벡터로 변환
- **인덱스 생성**: 빠른 검색을 위한 색인 생성
- **진행률 표시**: 실시간으로 처리 상황 확인

#### **지원 파일 형식**
- ✅ PDF 파일 (.pdf)
- ❌ 이미지 PDF (텍스트가 없는 스캔본)는 검색되지 않음

---

### 3️⃣ **검색하기**

#### **검색 방법**
1. **검색창에 내용 입력**
   ```
   예시:
   "계약 조건"
   "2024년 매출"
   "프로젝트 일정"
   "연락처 정보"
   ```

2. **검색 팁**
   - 🎯 **구체적인 단어**: "매출액" > "돈"
   - 🎯 **완전한 문장**: "계약서 조건" > "계약"
   - 🎯 **핵심 키워드**: "일정 관리" > "관리"

3. **"🔍 검색 시작" 버튼 클릭**

#### **검색 결과 이해하기**

각 검색 결과 카드에는 다음 정보가 표시됩니다:

```
┌─────────────────────────────────────────────────┐
│ 📄 문서명.pdf (p.15)           🔥 89%           │
│                                                 │
│ "계약 조건에 따라 2024년 1월부터 새로운        │
│  조건이 적용되며, 기존 계약서의 내용을..."      │
│                                                 │
│        💡 카드를 클릭하면 PDF가 열립니다         │
└─────────────────────────────────────────────────┘
```

#### **유사도 점수 의미**
- 🔥 **90-100%**: 거의 완벽한 일치
- ⭐ **80-89%**: 매우 관련성 높음  
- 👍 **70-79%**: 관련성 있음
- 📝 **60-69%**: 어느 정도 관련
- 📄 **60% 미만**: 낮은 관련성

---

### 4️⃣ **검색 결과 활용하기**

#### **PDF 열기**
- 결과 카드를 클릭하면 해당 PDF가 자동으로 열림
- Windows 기본 PDF 뷰어 또는 설치된 PDF 프로그램에서 열림

#### **결과 정렬**
- 결과는 유사도 순으로 자동 정렬됨
- 가장 관련성 높은 내용이 위에 표시

#### **여러 파일 검색**
- 모든 추가된 PDF에서 동시에 검색
- 파일별로 가장 관련성 높은 부분을 찾아서 표시

## 💡 효과적인 사용 팁

### 🎯 **검색 최적화**
1. **구체적인 검색어 사용**
   - ❌ "돈" → ✅ "매출액", "수익"
   - ❌ "사람" → ✅ "담당자", "연락처"

2. **문맥이 있는 검색**
   - ❌ "일정" → ✅ "프로젝트 일정"
   - ❌ "조건" → ✅ "계약 조건"

3. **여러 키워드 조합**
   - "2024년 매출 보고서"
   - "고객 만족도 설문 결과"

### ⚡ **성능 최적화**
- **많은 파일**: 한 번에 모든 파일을 추가하는 것이 효율적
- **큰 파일**: 처리 시간이 오래 걸릴 수 있음 (정상적인 현상)
- **메모리**: 대량의 PDF 처리 시 8GB 이상 RAM 권장

### 🗂️ **파일 관리**
- **파일 제거**: 목록에서 우클릭하여 제거 가능
- **파일 경로**: 한글 경로도 완벽 지원
- **파일 크기**: 큰 PDF도 처리 가능 (시간이 더 걸림)

## � 문제 해결

### ❓ **자주 묻는 질문**

**Q: 검색이 안 되는데요?**
- A: PDF가 텍스트를 포함하는지 확인하세요 (이미지만 있는 스캔본은 안됨)

**Q: 앱이 느려요**
- A: 큰 파일 처리 중일 수 있습니다. 진행률 바를 확인하세요

**Q: PDF가 열리지 않아요**
- A: PDF 뷰어가 설치되어 있는지 확인하세요

**Q: 검색 결과가 부정확해요**
- A: 더 구체적인 검색어를 사용해보세요

### ⚠️ **주의사항**
- 첫 실행 시 인터넷 연결 필요 (AI 모델 다운로드)
- 백신에서 차단되면 예외 처리 설정 필요
- 대용량 파일 처리 시 메모리 사용량 증가
